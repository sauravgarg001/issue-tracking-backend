Events to listen

verifyUser event
This event ("verifyUser") has to be listened on the user's end to verify user authentication.
user will only be set as online user after verification of authentication token.
auth-error@<authToken> event
This event ("online-users") has to be listened on the user's end to identify authentication error or breach in security.
The output will error message.
disconnect event
This event ("disconnect") has to be listened, when the user is disconnected.
